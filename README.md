# My dotfiles

Helps for installing a new environment and also for sharing with others :)

## Programs to install

- Text/code editor: Neovim 0.9
- Terminal emulator: Alacritty 0.12
- Shell: ZSH 5.9
- Shell prompt: Starship 1.16
- Nerd font: Iosevka Term 27.3
- Version control: Git 2.39
- CLI http client: Curl 7.88
- CLI grepper: Ripgrep 13.0
- Directory changer: Zoxide 0.9

## Languages/runtimes to install

- Rust
- Go
- NodeJS (via nvm)
